# Project Setup

## Frontend
1. Navigate to the `frontend` directory:
    cd frontend
    npm install
    npm run dev

## Backend
3. Navigate to the `backend` directory:
    cd backend
    npm install
    npm run db:reset
    npm start

## Testing
3. Navigate to the `backend` directory:
    cd backend
    npm test

If all tests pass, then all good! 
Use node.js to open a server!!
