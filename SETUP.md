# Project Setup

## Frontend
1. Navigate to the `frontend` directory:
    cd frontend
    npm install
    npm run dev

## Database
2. Navigate to the `backend` directory:
    cd backend
    mkdir database
    (now create a new file inside database, db.sqlite)

## Backend
3. Navigate to the `backend` directory:
    cd backend
    npm install
    npm start

## Testing
3. Navigate to the `backend` directory:
    npm test

If all tests pass, then all good!
